# http-server
Custom http server written in C++

## Instructions
Prerequisites:
- Make sure you have CMake installed.

Running:
- Clone, cd into this repo
- `$ git clone https://github.com/tarunbod/http-server.git && cd http-server`
- Generate makefile with CMake
- `$ cmake .`
- `make` it
- `$ make`
